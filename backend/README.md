# Backend Code

This folder contains all server-side logic, database operations, and API handlers.

## Subfolders

- **services/** - Business logic and database operations
- **api/** - API request handlers
- **database/** - Database query functions
- **utils/** - Backend utilities (email, logging, etc.)

## Example Files to Move Here

- `app/api/donate/` logic
- `app/api/profile/` logic
- Supabase database operations
- Email sending logic
- Payment processing logic
